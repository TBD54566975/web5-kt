CreateDidKeyOptions

class CreateDidKeyOptions(val algorithm: Algorithm = JWSAlgorithm.EdDSA, val curve: Curve = Curve.Ed25519) : CreateDidOptions(source)

Represents options for creating "did:key" DIDs with specified cryptographic configurations.

This class is designed to hold specific configurations, namely the algorithm and curve, during the creation of a new "did:key" DID using the DidKeyMethod implementation.
