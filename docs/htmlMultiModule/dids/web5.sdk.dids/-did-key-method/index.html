DidKeyMethod

object DidKeyMethod : DidMethod<CreateDidKeyOptions> (source)

Provides a specific implementation for creating and resolving "did:key" method Decentralized Identifiers (DIDs).

The "did:key" method is a specific DID method that is intended for use with DIDs that are entirely derived from a single public key, and it's described in detail in the W3C DID specification.

Usage Example:

val keyManager = InMemoryKeyManager()
val did = DidKeyMethod.create(keyManager)
